修改部署的在线 Tap 小游戏。

## About

本项目为 Dagou-Tap-New 的 Fork 版本。

相较于原项目，本版本主要进行了以下调整：

- 使用浏览器 `localStorage` 保存演奏设置和界面状态，不再依赖哔哩哔哩云存储或登录状态。
- 叮咚鸡音效与哈基米帝皇形象默认开放，无需观看作者视频解锁。
- 保留作者主页和开发视频入口，相关链接仅用于介绍与跳转，不再与功能开放条件关联。

配置仅保存在当前浏览器和当前站点中。清除浏览器站点数据、使用隐私浏览模式或更换浏览器后，
设置可能恢复为默认值。
